# 🌍 AI Multiverse

### 🤖 Chat with Multiple AI Personalities & Generate Stunning AI Images

An interactive AI-powered web application built with **Streamlit**, featuring **10 unique AI personalities** and an **AI Image Generator** powered by **Google Gemini AI** and **Pollinations AI**.

# 💡 How It Works

### AI Chat

1. Select an AI Personality
2. Type your message
3. Gemini AI generates a response
4. Conversation is saved automatically

---

### AI Image Generator

1. Enter your prompt
2. Choose an image style
3. Generate the image
4. Download with one click
